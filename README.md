# Vishal | Creative Developer Portfolio 🚀

A premium, modern portfolio website built with **React** and **Vite**, featuring a sleek dark-themed aesthetic, glassmorphism, and dynamic animations.

🔗 **Live Demo**: [portfolio-ver-2-eight.vercel.app](https://portfolio-ver-2-eight.vercel.app/)

## ✨ Key Features
- **Modern Tech Stack**: React 18, Vite, and Vanilla CSS.
- **Premium Aesthetics**: Deep dark mode with HSL-tailored gradients and `Outfit` typography.
- **Glassmorphism**: Elegant UI components with backdrop-blur and subtle borders.
- **Dynamic Background**: Interactive animated blobs for a "living" interface.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
- **Fast Performance**: Blazing fast load times thanks to Vite's optimized build process.

## 🛠️ Project Structure
- `src/components/`: Modular React components for every section.
- `src/index.css`: Centralized design system and utility classes.
- `src/App.jsx`: Main entry point and layout assembly.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/lucky20T/Portfolio-ver-2.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
