# Vishal | Aspiring Game Developer Portfolio 🚀

A premium, interactive portfolio website built with **React** and **Vite**, designed to showcase my journey as a Game Developer. Features a sleek dark-themed aesthetic, dynamic CSS animations, and a seamless user experience.

🔗 **Live Demo**: [portfolio-ver-2-eight.vercel.app](https://portfolio-ver-2-eight.vercel.app/)

## ✨ Key Features
- **Modern Tech Stack**: React 18, Vite, and highly customized Vanilla CSS.
- **Premium Aesthetics**: Deep dark mode with HSL-tailored gradients, glowing neon text shadows, and `Outfit` typography.
- **Interactive Visuals**: 
  - Dynamic "Falling Stars" background effect.
  - Animate-On-Scroll (AOS) for smooth section entrances.
  - Custom Typewriter text animation in the Hero section.
  - Epic 3D zooming and rotating popup modal for contact.
- **Glassmorphism Design**: Elegant UI components with backdrop-blur and subtle borders.
- **Functional Contact Form**: Integrated with FormSubmit to send messages directly to email without a backend.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.

## 🛠️ Project Structure
- `src/components/`: Modular React components (Hero, About, Projects, Skills, Contact, Navbar, Footer, StarsBackground).
- `src/index.css`: Centralized design system containing keyframe animations, glowing text classes, and utility styles.
- `src/Asset/`: Organized storage for certificates, project screenshots, and audio assets.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/lucky20T/Portfolio-ver-2.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
